This is the fifth application being built for FreeCodeCamp.  Please see the challenge [here](https://www.freecodecamp.org/challenges/build-a-wikipedia-viewer).

**User Stories**

* I can search Wikipedia entries in a search box and see the resulting Wikipedia entries.
